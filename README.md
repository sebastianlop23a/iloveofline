# Support Tool for Administrative Staff

Aplicativo desarrollado en Python para centralizar y optimizar la atención de problemas técnicos frecuentes en entornos administrativos.

## Problemática

El personal administrativo y asesores presentan incidencias recurrentes relacionadas con:

- Conexiones a redes privadas (VPN)
- Manejo de archivos pesados
- Conversión de formatos (especialmente PDF)
- Acceso y visualización de documentos

Estas situaciones generan alta carga de soporte técnico y repetición de solicitudes.

##  Objetivo

Desarrollar una herramienta que permita:

- Centralizar soluciones a problemas comunes
- Reducir solicitudes repetitivas al área de soporte
- Agilizar tiempos de respuesta
- Registrar y dar seguimiento a incidencias

##  Funcionalidades

- Soporte guiado para conexión a VPN
- Optimización y gestión de archivos pesados
- Conversión de archivos a PDF
- Visualización de documentos
- Registro básico de incidencias
- Automatización de soluciones frecuentes

## Tecnologías

- Python
- Scripts de automatización
- Procesamiento de archivos

##  Beneficios

- Reducción de carga operativa en soporte técnico
- Mejora en tiempos de atención
- Centralización del conocimiento técnico
- Mayor autonomía para el usuario final
